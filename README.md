# Axon AI

A modern, production-ready AI automation platform landing page built with **React** and **Vite**. The project showcases a premium SaaS user experience with responsive layouts, interactive pricing, adaptive components, and a scalable design system.

---

## 🚀 Features

* Modern AI SaaS Landing Page
* Fully Responsive Design
* Adaptive Bento Grid → Accordion Layout
* Dynamic Multi-Currency Pricing
* Monthly & Annual Billing Toggle
* Interactive Navbar
* Hero Section
* Feature Showcase
* Pricing Plans
* Footer with Social Links
* Reusable Components
* Scalable Design System
* Optimized Performance
* Accessibility Friendly

---
